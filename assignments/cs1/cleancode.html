Cleaner code

Assignment overview

In this assignment, you will practice writing functions that are so clear they sound like English. There are two pieces to your homework:

  1. You will implement a couple more complex functions, writing helper functions along the way.

  2. You will revise a messy piece of code I have written that causes a picture of Justin Bieber to rotate across the screen endlessly.

Writing clean code

Please design the following functions, in a file called YourNameCleanFunctions.rkt. For each one, make sure to choose names and design helper functions so the function is easy to understand.

1Love letter

Design a function that generates a love letter. You can choose what pieces of information (arguments) you’d like to accept: recipient’s name, gender pronouns, best attributes, worst attributes... the possibilities are endless. The output should be a string. (It’s OK if the letter is all on one line. We haven’t yet learned how to make strings with multiple lines in them.)

2Movie poster

Design a function that creates a movie poster. You can choose what pieces of information (arguments) you’d like to accept: movie title, names of stars, theme color, main poster image, tagline... the possibilities are endless. Return an image of the poster.

3Falling things

Design a function for an animation (that is, a Number -> Image function) in which at least five things (circles, squares, etc.) fall down the screen, at different speeds and starting from different heights. (Note: if I were to ask you to add a sixth or seventh object, you shouldn’t need to add new helper functions.)

Extra Challenge: Clean up an old submission

Select an extra-challenge problem from one of the previous two homeworks (Designing functions or Images and animation) and use your newfound clean-code skills to write a clean version of it. You may find that previously difficult problems become much easier when you are able to “work backward” and keep a wishlist of helper functions.

Refactoring messy code

In the tech community, cleaning up messy code by reorganizing it is called “refactoring.”

4Messy Bieber

Download the attached MessyCode.rkt file, and open it. You may want to run (animate messy-animation) and turn on some of your favorite Justin Bieber music while you watch. Your goal is to clean up this messy code as best you can. You can define individual values that are used a lot. You can define helper functions. The goal is to make the code as readable as possible. Upload your revised version with the file name YourNameRefactoredBieber.rkt. (Use the “Save Definitions As...” menu item, under the File menu, to save a copy of your code with a new filename.)

Useful built-in functions

You may find the following functions useful in completing this assignment:

  • empty-scene : Number Number -> Image
    Creates an empty rectangular scene of the given width and height, in which to place other images.

  • place-image : Image Number Number Image -> Image
    Places the first image on top of the second image to obtain a new image. The image is placed so that its center appears x pixels over and y pixels down from the top left corner of the screen, where x and y are the two numerical arguments.

  • overlay : Image Image -> Image
    Places the first image on top of the second image, so that the centers of each image are aligned. (That is, the first image is placed in exactly the center of the second.)

  • text : String Number String -> Image
    Creates an image out of a string. The first string argument determines what text to display; the number and the second string argument determine the font size and color, respectively.
