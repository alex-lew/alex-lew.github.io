Getting started with Go

Assignment overview

The goal of this assignment is to get you up and running with Go. When using Go, there is no single integrated tool that allows you to write, test, and run your code as easily as you could with DrRacket. Instead, you’ll install several things: the Go compiler (to translate Go code into machine code), a text editor (to write your Go code), and some useful tools that will help you locate errors and format your work. Then, in order to make sure everything is installed correctly, you’ll write a simple program introducing yourself.

Setting up

1Install the tools

You’ll need to install two things for this semester in CS 2.

  1. The Go compiler. Go to http://golang.org and download the version for your operating system. Run the installer.

  2. A text editor. I recommend Visual Studio Code. Once you’ve downloaded it, you’ll need to run a few commands.

    Press Cmd+Shift+P (on a Mac) or Ctrl+Shift+P (on Windows) to open the command prompt. Type “Shell” and select “Shell Command: Install code command in PATH.” (This may not work on Windows; that’s OK.)

    Once this has installed, go to the Package view (click the square icon on the lefthand bar, right under the “debug” icon), and find the Go package (the description says, “Rich Go language support for Visual Studio Code”). Install it, then click “Reload” to complete the installation.

    Open the command prompt again (Cmd or Ctrl + Shift + P), and type “Go install”; select “Go: Install/update tools.” This will install a number of useful tools for you.

You should be ready to go!

2Set up your Go directory

To use Go, your files must be organized in a very specific way. To set up, do the following:

  1. If on Windows, install Git for Windows (https://git-for-windows.github.io).

  2. Open your system command line terminal. On Mac, this is an app called “Terminal.” On Windows, please use “Git Bash,” which should have been installed in the previous step.

  3. Once there, type cd ~ and press Enter. This will move you to your home directory, or home folder.

  4. Type mkdir -p go/src/sample. Press enter. This creates a folder called “go” inside your home directory, a folder called “src” inside that, and a folder called “sample” inside that. For every Go program that you write, you will create a new folder inside the “src” folder. “sample” is the program you will write now.

  5. Type code go/src/sample, which will open Visual Studio Code to the new directory you’ve created. Use the “new file” icon in the sidebar to create a file called sample.go inside the directory. Type the following:

  6. Press Ctrl+` (that’s the backtick character, same key as ~) to open the Terminal inside Visual Studio Code. Type go build, and press enter. This will compile your program into a machine-code file called sample (on Windows, sample.exe). You can run it by typing ./sample (or ./sample.exe on Windows) and pressing enter. You should see “Hello, world!” printed to the screen.

3A program about yourself

Create another folder in the same way you created sample, this time called aboutme. Open it in Visual Studio Code. Create a file called YourNameAboutMe.go, and inside, write a program that prints out the answers to the following questions:

  1. How was your summer? What was the best or worst experience you had?

  2. Do you see any way these instructions be better? Any suggestions?

  3. Is there anything else you’d like to say / like me to know?
