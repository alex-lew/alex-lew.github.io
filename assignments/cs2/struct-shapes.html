Shapes with structs

Assignment overview

In this assignment, you will create a simple “drawing” library for the Terminal. In the process, you will practice creating structure types and writing methods for them!

Problems

1A type for positions

Create a structure type for storing an (x, y) cooridnate pair, using float64s. Give it the following methods:

  • (p posn) moveBy(other posn), which returns a position moved a certain amount in x and y directions, as determined by the argument (also a position).

  • (p posn) distanceFrom(other posn), which uses the distance formula to return a float64 representing how far p is from the argument position.

  • (p posn) multiplyBy(amount float64), which multiplies each coordinate by the given amount.

  • (p posn) stepToward(other posn, amount float64), which returns a position moved toward another position by a certain amount. Hint: Use other.x - p.x, other.y - p.y to get the amount you’d have to travel from p to other in one step. You can then divide this by the distance between p and other, and multiply by amount, to get the amount you have to move in each direction. This can then be passed to moveBy.

  • (p posn) roundedX(), which returns the x coordinate, rounded to the nearest integer.

  • (p posn) roundedY(), which returns the y coordinate, rounded to the nearest integer.

2Rectangle

Create a rectangle type, which stores a position at which it starts, a width, and a height.

It should have moveBy and stepToward methods that return new rectangles with the appropriate methods called on the rectangle’s position. It should also have a draw method, which takes no arguments and draws this rectangle on the screen, using | and – characters. If the rectangle’s (rounded) position is (5, 7), there should be 5 columns of spaces and 7 rows of spaces before the rectangle begins. Furthermore, so that this will work well with your next problem, please draw at least 24 rows–if the rectangle ends before the 24th row, just add empty rows after you’ve drawn the rectangle.

3Flipbook animation

Design an “animation” of sorts, by doing the following:

  1. Write a function rectangleForFrame(frame int), which takes in a “frame number” of the animation, and returns a rectangle that should be drawn at that frame. You can choose to make whatever animation you want–perhaps the rectangle moves toward a point, or maybe it grows and then shrinks, and so on.

  2. Write a function animate(anim func (int) rectangle, numFrames int), which takes in an animation function (like the one you wrote above), and animates it for numFrames frames. To do this, call the rectangle’s draw method on successively higher frame numbers, starting at 0 and going to numFrames. Between each frame, use fmt.Scanln(), which will wait for the user to press Enter.

  3. Call your function from main.
